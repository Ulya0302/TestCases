1. Анимация зеленой галочки
Шаги:
Пользователь выбирает вакансию 
Ождиаемый результат:
Поле загорается зеленым

2. Создание формы без выбора значения в поле "Вакансия"
Шаги:
Пользователь заполняет форму правильно, кроме поля "Вакансия"
Пользователь не выбирает вакансию 
Ождиаемый результат:
Поле "Вакансия" подсвечено красным


3. Нажатие на поле "Вакансии"
Шаги:
Пользователь нажимает на поле "Вакансии"
Ождиаемый результат:
Появляется выпадающий список

4. Проверка полноты в списке вакансий
Предусловия:
Пользователь устраивается на конкретную профессию из предложенных
Шаги:
Пользователь нажимает на поле "Вакансии"
Ождиаемый результат:
Все предложенные вакансии присутствуют в списке и отсутствуют лишние

5.Отсутствие выбора вакансии
Шаги:
Пользователь не выбирает вакансию
Пользователь делает корректный ввод всех остальных полей
Ождиаемый результат:
Поле Вакансия подсвечено красным, поле обязательно к заполнению

6.Прокрутка выпадающего списка
Шаги:
Пользователь нажимает на поле "вакансии"
Пользователь прокручивает этот список
Ождиаемый результат:
Появляется выпадающий список
Список ПРОКРУЧИВАЕТСЯ (это важно(ОЧЕНЬ))

8.Закрытие выпадающего списка
Шаги:
Пользватель нажимает на поле "вакансии"
Пользователь повторно нажимет на поле "вакансии"
Ождиаемый результат:
Выпадающий список открылся
Выпадающий список закрылся
