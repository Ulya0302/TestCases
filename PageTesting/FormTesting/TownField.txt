1. Заполнение поля "Город" латинскими символами 
Предусловия:
Пользователь - иностранец
Шаги:
Пользователь вводит в поле "Город" английские символы
Ождиаемый результат:
Ввод разрешен, поле помечается зеленым 

2. Заполнение поля "Город" цифрами
Шаги:
Пользователь вводит в поле "Город" цифры
Ождиаемый результат:
Поле "Город" помечается красным, некорректный ввод

3.Ввод пустого значения в поле Город
Шаги:
Пользователь не вводит город
Ождиаемый результат:
Поле город подсвечено красным, поле обязательно к заполнению

4. Проверка дефиса в поле город
Шаги:
Пользователь вводит в поле Город название города с символом "-"
Ождиаемый результат:
Поле подсвечивается зеленым

5. Составное название города
Шаги:
Пользователь вводит в поле Город название города, состоящее из нескольких слов (в т.ч. через дефис)
Ождиаемый результат:
Поле подсвечивается зеленым, корректный ввод

6. Анимация зеленой галочки
Шаги:
Пользователь вводит корректную информацию в поле Город
Ождиаемый результат:
Поле загорается зеленым

7. Проверка работы крестика
Шаги:
Пользователь вводит неверную информацию в поле e-mail 
Пользователь нажимает на красный крестик, появившийся на поле
Ождиаемый результат:
Поле подсвечивается красным
Поле очищается

8. Анимация изменения стандартного текста в поле
Шаги:
Пользователь нажимает на поле Город
Ождиаемый результат:
Надпись "Город*" на поле уменьшается или исчезает
