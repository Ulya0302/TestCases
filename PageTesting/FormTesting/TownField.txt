1. Корректный ввод в поле "Город", анимация зеленой галочки
Приоритет: 1
Шаги:
Пользователь вводит корректную информацию (1 или более слово на кириллице и/или латинице, не содержит цифр и любых других символов, кроме дефисы), примеры:
1) Москва 
2) Ростов-на-Дону - содержит дефис
3) London - латинские символы
4) Нижний Новгород - состоит из нескольих слов
Результат:
Поле город подсвечивается зеленым, корректный ввод

2. Некорректный ввод в поле "Город"
Приоритет: 2
Шаги:
Пользователь вводит в поле "Город" некорректную информацию, примеры: 
1) Нижний Новгород1 - содержит цифры
2) Нижний Новгород! - содержит символ (вместо ! любой другой символ, не являющийся буквой)
Ождиаемый результат:
Поле "Город" помечается красным, некорректный ввод, надпись "Использованы запрещенные символы, введите корректное название города"

3.Ввод пустого значения в поле Город
Приоритет: 2
Шаги:
Пользователь нажимает на поле город
Пользователь нажимает на любое место вне поля город, оставляя поле пустым
Ождиаемый результат:
Поле город подсвечено красным, поле обязательно к заполнению, некорректный ввод "Поле обязательно для заполнения"

4. Анимация изменения стандартного текста в поле
Приоритет: 3
Шаги:
Пользователь нажимает на поле Город
Ождиаемый результат:
Надпись "Город*" на поле уменьшается или исчезает

5. Проверка работы крестика
Приоритет: 4
Шаги:
Пользователь вводит неверную информацию в поле Город
Пользователь нажимает на красный крестик, появившийся на поле
Ождиаемый результат:
Поле подсвечивается красным
Поле очищается
Курсор перемещаеся в поле для ввода
