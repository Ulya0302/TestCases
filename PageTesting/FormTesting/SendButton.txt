1. Проверка активирования кнопки "Отправить"
Шаги:
Пользователь заполняет форму правильно
Пользователь ставит галочку на согласие об обработке данных
Ождиаемый результат:
Кнопка "Отправить" стала активной

2. Проверка работоспособности кнопки "Отправить" 
Шаги:
Пользователь заполняет форму правильно
Пользователь ставит галочку на согласие об обработке данных
Пользователь нажимает на кнопку "Отправить"
Ождиаемый результат:
Кнопка "Отправить" стала активной
Появление надписи об успешной отправке

3. Проверка активирования кнопки "Отправить" [2]
Шаги:
Пользователь заполняет форму правильно
Пользователь не ставит галочку на согласие об обработке данных
Ождиаемый результат:
Кнопка "Отправить" неактивна

4. Реакция кнопки "Отправить" на изменение корректных данных на некорректные
Шаги:
Пользователь корректно заполняет форму и ставит галочку на согласие обработки данных
Пользователь делает некорректный ввод в одном из поле или убирает галочку
Ождиаемый результат:
Кнопка "Отправить" активна
Кнопка "Отправить" перестает быть активной
