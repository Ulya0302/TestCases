1.Создание пользователя с ФИО на латинице
Предусловия:
Пользователь - иностранец
Шаги:
Пользователь вводит ФИО латинскими символами
Пользователь заполняет остальную часть формы правильно
Результат:
Поле подсвечивается зеленым (принято)
Заявка отправлена

2.Создание формы с ФИО без отчества
Предусловия:
Пользователь - иностранец
Шаги:
Пользователь вводит ФИО без отчества
Пользователь заполняет остальную часть формы правильно
Результат:
Поле подсвечивается зеленым (принято)
Кнопка "Отправить" доступна

3.Заполнение ФИО тольно фамилией
Шаги:
Пользователь вводит в поле "ФИО" одно слово
Результат:
Поле "ФИО" помечается красным (некорректный ввод)

4.Нажатие на поле ФИО
Шаги:
Пользователь нажимает на поле ФИО
Результат:
Курсор помещается в поле, разрешен ввод


5.Ввод пустого значения в поле ФИО
Шаги:
Пользователь не вводит ФИО
Пользователь делает корректный ввод всех остальных полей
Результат:
Поле ФИО подсвечено красным, поле обязательно к заполнению
Кнопка "Отправить" недоступна

6.Сложные фамилии
Шаги:
Пользователь вводит составные имя/фамилию (через "-")
Результат:
Поле ФИО - зеленое, корректный ввод

7.Цифры в поле ФИО
Шаги:
Пользователь вводит цифры в поле "ФИО"
Результат:
Поле загорается красным

8.Анимация зеленой галочки
Шаги:
Пользователь вводит корректную информацию в поле ФИО
Результат:
Поле загорается зеленым
